# 插件抽屉（plugin-hub）

Hana 插件抽屉。管理顶栏插件置顶/收纳、文件夹分类、主题切换。

## 安装

1. 下载 [最新 Release](https://github.com/hayou2002/plugin-hub/releases/latest) 的 zip
2. 打开 Hana → 插件管理
3. 拖入 zip 安装，重启 Hana

## 用法

### 浏览模式（默认）
- 安装后顶栏出现「抽屉」标签
- 隐藏的插件按文件夹分组展示，点击即可打开
- 右上角 🎨 按钮切换主题配色

### 管理模式
- 点击右上角 ⚙ 管理进入
- 通过开关控制每个插件的显示/隐藏
- 创建文件夹对插件进行分类
- 一键「全部收进」或「全部置顶」

### 主题
- 6 个预设主题：极光、落日、海洋、樱花、森林、午夜
- 选择自动保存，下次打开自动应用

## 文件结构

```
plugin-hub/
  manifest.json          ← 插件配置
  index.js               ← 入口
  routes/
    hub.js               ← 路由 + HTML 模板
    patcher.js           ← 补丁引擎（保留备用）
  assets/
    hub.css              ← 样式（液态玻璃 + 主题变量）
    hub.js               ← 前端逻辑（浏览/管理/主题）
  tools/
    install-patch.js     ← 补丁安装工具（保留备用）
  README.md
```

## 兼容性

- `minAppVersion`: 0.82.0
- 支持 Windows / macOS / Linux
- 自动适配明暗主题
