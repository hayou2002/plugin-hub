# 插件抽屉（plugin-hub）

Hana 插件抽屉。管理顶栏插件置顶/收纳、文件夹分类、主题切换。**安全版**：不修改 Hana 核心文件，启动时自动隐藏顶栏 ▼ 按钮。

## 安装

1. 下载 [最新 Release](https://github.com/hayou2002/plugin-hub/releases/latest) 的 zip
2. 打开 Hana → 插件管理
3. 拖入 zip 安装，重启 Hana

## 用法

### 浏览模式（默认）
- 安装后顶栏出现「抽屉」标签
- 隐藏的插件按文件夹分组展示，点击即可打开
- 右上角 🎨 按钮切换主题配色
- **顶栏 ▼ 按钮自动隐藏**（插件启动时注入 CSS，不修改种子文件）

### 管理模式
- 点击右上角 ⚙ 管理进入
- 通过开关控制每个插件的显示/隐藏
- 创建文件夹对插件进行分类
- 一键「全部收进」或「全部置顶」

### 主题
- 6 个预设主题：极光、落日、海洋、樱花、森林、午夜
- 选择自动保存，下次打开自动应用

## 安全说明

v1.6.0 彻底移除了危险的 renderer JS 桥接补丁。▼ 按钮隐藏通过插件启动时向已解压的 renderer CSS 文件追加一条规则实现：

- ✅ 不修改 `seed/` 目录下的 tar.gz 种子文件
- ✅ 不触发 SHA256 校验错误
- ✅ 每次 Hana 启动自动注入
- ✅ 版本更新后自动适配新 CSS 文件
- ✅ 多条 CSS 选择器作为 fallback

## 文件结构

```
plugin-hub/
  manifest.json          ← 插件配置
  index.js               ← 入口（启动时注入 CSS）
  routes/
    hub.js               ← 路由 + HTML 模板
    patcher.js           ← 旧补丁引擎（保留参考，不再使用）
  assets/
    hub.css              ← 样式（液态玻璃 + 主题变量）
    hub.js               ← 前端逻辑（浏览/管理/主题）
  tools/
    install-patch.js     ← 补丁安装工具（安全版，CSS 注入）
  README.md
```

## 兼容性

- `minAppVersion`: 0.82.0
- 支持 Windows / macOS / Linux
- 自动适配明暗主题
